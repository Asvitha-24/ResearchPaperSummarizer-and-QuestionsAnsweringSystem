## ğŸ” What Each Command Does

### `python main.py preprocess`
**Input:** Raw CSV with text column  
**Process:** Cleans text, removes URLs/emails, tokenizes  
**Output:** CSV with:
- `cleaned_{column}` - Cleaned text
- `word_count` - Number of tokens
- `sentence_count` - Number of sentences

### `python main.py index`
**Input:** Processed CSV with cleaned text  
**Process:** Creates embeddings/TF-IDF vectors  
**Output:** Pickled retriever in `checkpoints/`

### `python main.py summarize`
**Input:** Processed CSV  
**Process:** BART abstractive summarization  
**Output:** CSV with summary column

### `python main.py qa`
**Input:** Processed CSV + retriever index  
**Process:** Retrieves top-k docs â†’ Extracts answers  
**Output:** Console display + optional JSON

### `python main.py demo`
**Input:** Raw CSV at `data/raw/arXiv Scientific Research Papers Dataset.csv`  
**Process:** Preprocess â†’ Index â†’ QA samples  
**Output:** Multiple files + console results
