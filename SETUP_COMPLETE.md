## CLI Features

### 5 Main Commands

1. **`preprocess`** - Clean & tokenize raw data
   - Removes URLs, emails, special chars
   - Outputs word/sentence counts

2. **`index`** - Build retrieval index
   - TF-IDF, semantic, or hybrid
   - Saves as pickled object

3. **`summarize`** - Generate summaries
   - Uses BART transformer
   - Customizable length

4. **`qa`** - Answer questions
   - Retrieves relevant docs
   - Extracts answers with scores
   - Optional JSON export

5. **`demo`** - Run full workflow
   - Automated end-to-end
   - Perfect for testing

## Troubleshooting

### CLI not responding
```powershell
# Check Python is correct
python --version

# Check dependencies
pip list | findstr transformers
pip list | findstr nltk
```

### "File not found" error
- Check file path is correct
- Use absolute paths if in doubt
- Check column names match CSV

### Slow processing
- First run downloads models (~1-2GB)
- Use TF-IDF for instant results
- Check GPU is available

### Out of memory
- Reduce dataset size
- Use TF-IDF instead of semantic
- Close other applications
