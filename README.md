# Research Paper Summarizer & Question Answering System

## **Overview**
This project implements a **Research Paper Summarizer** and **Question Answering (QA) System** using **Python**, **Machine Learning (ML)**, and **Natural Language Processing (NLP)** techniques. The system can:
- Summarize research papers into concise versions.
- Answer research-related questions based on document content.

## **Features**
- **Text Summarization**: Automatically generates summaries of research papers.
- **Question Answering**: Answer specific questions from a collection of documents.
- **Data Handling**: Uses a dataset of research papers and metadata for model training and evaluation.

## **Installation**

### **Prerequisites**
Make sure you have the following installed:
- **Python 3.6+**
- **Poetry** (for managing dependencies)
